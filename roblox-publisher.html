ℹ️ How to use
  1. Generate an API key in Roblox Creator Hub with Place Files: Write permission.
  2. Paste and save your key above — stored only on this device.
  3. Enter your Universe ID and Place ID from Creator Hub.
  4. Select your .rbxl or .rbxlx place file.
  5. Tap Publish to Roblox!

API key stored locally via IndexedDB — never sent anywhere except Roblox.
